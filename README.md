# 🎸 Live Instrument Detector

This app uses Streamlit and WebRTC to capture live audio and classify musical instruments in real-time.

## 🚀 How to Run Locally
1. **Clone the repo:**
   `git clone https://github.com/YOUR_USERNAME/instrument-classifier.git`
2. **Install dependencies:**
   `pip install -r requirements.txt`
3. **Run the app:**
   `streamlit run app.py`

## 🛠️ Tech Stack
- **Streamlit**: Web Interface
- **Librosa**: Audio feature extraction (MFCCs)
- **Streamlit-WebRTC**: Real-time browser audio streaming
